## 설치 | Installation
1. Chrome 브라우저 설치

2. Python 3.10 설치

3. 폴더 생성

```
$ mkdir git
```

4. 생성한 폴더로 이동 

```
cd git
```

5. git clone을 사용하여 local로 레포지토리 가져오기

```
$ git clone https://github.com/JaehyoJJAng/Instagram-Image-Crawling.git
```

6. 필수 라이브러리 설치

```
$ cd Instagram-Image-Crawling
$ pip install -r requirements.txt
```

7. 프로그램 실행

```
$ python main.py 
```
